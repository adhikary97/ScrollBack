## Rotary Encoder -> ESP32 Wiring
- COMB -> GND
- SW5 -> D12
- SW4 -> D14
- SW3 -> D27
- SW2 -> D26 
- SW1 -> D25
- COMA -> GND
- ENCB -> 33
- ENCA -> 32
